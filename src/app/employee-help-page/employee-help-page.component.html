AVS Fitness Gym System Help Guide

Home

Welcome to AVS Fitness Home. This is the landing page for all employees. It provides general information focused on conveying the ethos of what it means to be an employee at AVS Fitness.

  • Click the "Read More" buttons to expand the paragraph text information.
  • Click on any of the bottom cards to go to the relevant pages where you will find further information on the subject matter.

Booking Manager

The Booking Manager is the primary hub for facilitating and provisioning member bookings. The main objectives include:

  • Workouts:Use the create buttons and subsequent forms to create new workouts or lesson plans. use control buttons and click on them to start creating, updating, viewing, and deleting booking information
  • Lesson Plans:Lesson plans are comprised of workouts, so always create new workouts first.
  • Bookings:Bookings are where lesson plans will need created booking slots for members to book. Attendance lists can be generated in PDF format.

Equipment Manager

This interface is primarily concerned with creating, updating, viewing, and deleting new and existing equipment items for the gym.

  • Click into the "Equipment Manager" link and then into the "Equipment" card tab to find relevant control buttons.
  • Fill in the associated forms and dialogues that appear to complete, view, and interact with equipment management tasks.
  • Part of the AVS Fitness employee job roles includes conducting an equipment inspection.
  • View all equipment inspections in the "Inspections" tab, where you can view the details in a table format.

Gym Manager

This interface is the primary hub for facilitating a variety of gym management tasks and responsibilities.

Settings

This section is primarily concerned with setting new business rules and enforcing them throughout the system.

  • Deletion Settings: Use the configurable timer to set when users will get deleted. Click "Update Setting" to change the length of time and unit. This affects when a user will be deleted once activated.
  • VAT Settings: Configure and set the VAT settings and percentage.
  • Discount Settings: Set the discount percentage used for products that members will buy.
  • Contract Settings: Configure and set the deletion period for terminated contracts after a specified period. Click "Update Setting" to confirm the change.

Dashboard

This section is where administration will be able to generate specified reports (e.g., Member, Financial, Sales, Audit Trail, and Inspections Reports).

  • The landing page displays a graph with statistics for a specified business area. This can be filtered monthly, weekly, or daily.
  • Click "Generate Report," select a report type, and indicate the reporting period (1 month, 3 months, 6 months, or 12 months). A PDF will be generated.

User Manager

This interface consists of several tabs, primarily focused on user configurations, member management, and user role enforcement:

  • Configure User Types: For administration to navigate through and create, edit, delete, and view user types which facilitate user role permissions. Use the labeled control buttons to manage user types.
  • Users: Administration can configure automated deletion for users and manage a list of all users in the system. Use manual control buttons to reactivate and deactivate users' ability to log in and use the system.
  • Contract Manager: Create, govern, and provision contract management for members by uploading contracts, creating new contracts, approving pending memberships, and viewing contract subscription statistics.

Audit Trail

This interface consists of a table log with all user actions logged and searchable:

  • Administration can navigate through and view all types of system interactions, including the type of action, time and date, the user who performed the action, and the details of the action.

Inventory Manager

This interface consists of several tabs, including the "Supplier" card tab:

  • Suppliers: Add or view suppliers, place supplier orders, and manage inventory.
  • Products: Add, create, update, and delete supplier orders. These orders become part of the inventory available for management and member purchases.
  • Inventories: Add, create, update, and delete inventory items. Administration can also conduct an inventory inspection as part of employee responsibilities.
  • Inspections: Created inspections can be viewed for reporting purposes.

Member Manager

This interface is concerned with governing member-related activities:

  • Subscriptions: Manage member subscriptions, block members who have not met monthly payments, and reactivate them.
  • Cancellations: Approve contract terminations and inspect the cancellation reason.
  • Rewards: Configure, set, and post rewards for qualifying members.
  • Orders: View member orders and complete the collection process.
  • Payments: View all payments made by members and export the information as an Excel spreadsheet.

Profile

This interface is primarily concerned with managing yourself and your details.

  • Updating Personal Details: If any of your basic user details require a change, navigate to your profile, click the edit button, change your details, and save them.

Frequently Asked Questions (FAQs)

  • How do I navigate the AVS Fitness system? Use the main menu to access different sections such as Home, Booking Manager, Equipment Manager, etc.
  • What if I encounter an issue? Contact system support or refer to this help guide for assistance.
  • How do I generate reports? Go to the Dashboard section, select the report type and reporting period, and click "Generate Report."
  • I forgot my password: Navigate to the login screen and click "Forgot Password" fill in your current user email and click the reset button an email will be sent out to the provided email address where there will be a link to navigate you to the Change Password screen. There you will be able to create a new password and use the new password to login .
