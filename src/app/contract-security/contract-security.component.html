What is contract security? If an employee accidentally uploads an incorrect PDF, they will navigate to "Uploaded Incorrect PDF File" where they will enter the member's name and surname and the password. If the name and surname of the member are correct and the password matches, they will successfully remove the contract. As an owner, you have the added ability to set the password to facilitate this process. Make sure it is at least 6 digits long.
